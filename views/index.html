URL Shortener Microservice

POST /api/shorturl/new

Creates a short URL for a given valid URL.

Try it out

Example Request:

POST /api/shorturl/new 
{
  "url": "https://www.freecodecamp.org"
}

Response:

{
  "original_url": "https://www.freecodecamp.org",
  "short_url": 1
}

Using Short URL:

GET /api/shorturl/1

Redirects to: https://www.freecodecamp.org

Invalid URL Example:

POST /api/shorturl/new 
{
  "url": "invalid-url"
}

Response:

{
  "error": "invalid url"
}
